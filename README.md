# 🥐 FyL - Pastelería Premium

![FyL Header](https://raw.githubusercontent.com/BobCatChyvo/fyl_lp/main/public/header.jpg) 

> Focus Your Life - Una experiencia culinaria diseñada para los paladares más exigentes.

Esta es una **Landing Page de Pastelería Premium** construida con tecnologías modernas para ofrecer una experiencia de usuario fluida, elegante y visualmente impactante.

## ✨ Características Principales

- **Arquitectura Next.js 14**: Aprovechando el App Router para una navegación óptima.
- **Diseño Estético Premium**: Tipografías seleccionadas (Inter & Playfair Display) y paleta de colores sofisticada.
- **Micro-animaciones**: Transiciones suaves y efectos de hover integrados con Tailwind CSS y Headless UI.
- **Gestión de Estado con Zustand**: Carrito de compras reactivo y eficiente.
- **Diseño Responsivo**: Adaptado perfectamente a móviles, tablets y escritorio.
- **Optimización SEO**: Estructura semántica, metaetiquetas y accesibilidad mejorada.

## 🚀 Tecnologías

| Herramienta | Uso |
| :--- | :--- |
| **Next.js 14** | Framework de React para el frontend. |
| **TypeScript** | Tipado estático para un desarrollo robusto. |
| **Tailwind CSS** | Estilizado moderno y utilitario. |
| **Zustand** | Gestión de estado ligera para el carrito. |
| **Lucide React** | Set de iconos vectoriales elegantes. |
| **GitHub Actions** | CI/CD para despliegue automático en GitHub Pages. |

## 🛠️ Instalación y Desarrollo

Para ejecutar este proyecto localmente, sigue estos pasos:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/BobCatChyvo/fyl_lp.git
   cd fyl_lp
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción:**
   ```bash
   npm run build
   ```

## 📦 Despliegue

Este proyecto está configurado para desplegarse automáticamente en **GitHub Pages** a través del flujo de trabajo definido en `.github/workflows/deploy.yml`. 

Cada `push` a la rama `main` activará una construcción automática y actualizará la versión en línea.
